## Come Convertire MD in PDF
Se si preferisce leggere su pdf, suggerisco di installare `pandoc`

`sudo apt install pandoc`

e poi utilizzare il comando

`pandoc --from=gfm --to=pdf <file_da_convertire> -o <file_di_output>.pdf`
